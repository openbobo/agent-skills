## üìÅ Categories

### [GitHub CLI](./github.md)
- `gh run view <id> --log-failed` - Show only failed CI/CD steps
- `--json` + `--jq` for structured API output
- `gh api repos/owner/repo/pulls/55 --jq '.title, .state, .user.login'`

### [tmux](./tmux.md)
- Socket isolation via `CLAWDBOT_TMUX_SOCKET_DIR`
- `PYTHON_BASIC_REPL=1` for Python REPL automation
- `capture-pane -p -J` joins wrapped lines
- `wait-for-text.sh` helper for regex polling
- Detect completion via `‚ùØ` or `$` prompts

### [Coding Agent](./coding-agent.md)
- `workdir:<path>` pattern to focus agents
- `--yolo` / `--full-auto` modes
- Process tool: `process action:log/poll/write/kill`

### [Google Workspace (gog)](./gog.md)
- Gmail: `gog gmail search 'newer_than:7d' --max 10`
- Calendar: `gog calendar events <id> --from <iso> --to <iso>`
- Drive: `gog drive search "query" --max 10`
- OAuth setup: `gog auth add <email> --services gmail,calendar,drive`

### [Obsidian](./obsidian.md)
- Vaults are plain folders (edit .md directly)
- `obsidian-cli move` auto-updates ALL wikilinks
- Config: `~/Library/Application Support/obsidian/obsidian.json`

### [MCP (mcporter)](./mcporter.md)
- `mcporter list` - List servers & schemas
- `mcporter call <server>.<tool> <args>`
- `--output json` for scripting
- `mcporter generate-cli` for TypeScript wrappers

### [Cron Jobs](./cron.md)
- System events for scheduled tasks
- Heartbeats for periodic checks
- Separate from main session history
