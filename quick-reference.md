# ⚡ Quick Reference

## GitHub
| Command | Description |
|---------|-------------|
| `gh run view <id> --log-failed` | Show only FAILED CI steps |
| `gh api ... --jq '.field'` | Extract specific fields |

## tmux
| Pattern | Description |
|---------|-------------|
| `PYTHON_BASIC_REPL=1` | Required for Python REPL |
| `capture-pane -p -J` | Join wrapped lines |
| `❯` or `$` | Detect completion |

## Coding Agents
| Pattern | Description |
|---------|-------------|
| `workdir:/path` | Focus agent on directory |
| `process action:poll` | Check if done |

## Gmail (gog)
| Command | Description |
|---------|-------------|
| `gog gmail search 'newer_than:1d'` | Recent emails |
| `gog gmail send --to x --subject y` | Send email |

## Obsidian
| Command | Description |
|---------|-------------|
| Vault = folder | Edit .md files directly |
| `obsidian-cli move` | Auto-updates wikilinks |

## MCP
| Command | Description |
|---------|-------------|
| `mcporter list` | List servers |
| `mcporter call <tool> <args>` | Call tool |
| `--output json` | Script-friendly output |
